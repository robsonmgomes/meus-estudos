Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase tenho um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica)

Nesta frase, temos um termo com ênfase usando a tag EM (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK. Dentro da tag MARK, podemos utilizar um parâmetro CSS através do "style" para mudar a cor da marcação usando a propriedade "background-color"

Texto grande e pequeno

Estamos criando um texto grande utilizando a tag BIG (obsoleto) e um texto pequeno utilizando a tag SMALL

Texto deletado

podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Para isso, utiliza-se a tag DEL

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. Para isso, utiliza-se a tag INS

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, utilizar a tag SUP

Texto subscrito

Para inserir coisas do tipo H2O, utilizar a tag SUB
